# Azure Labs

This repository contains hands-on labs and experiments for learning Microsoft Azure cloud infrastructure.

The goal is to build practical experience with core Azure services through real-world scenarios.

## Topics Covered

- Virtual Networks
- Network Security Groups
- Virtual Machines
- Storage Accounts
- Azure Monitor
- RBAC & Azure Policy
- Resource Locks
- Cloud Governance

## Lab Categories

### Networking

- Virtual Network setup
- VNet Peering
- Network Security Groups

### Compute

- Virtual Machine deployment
- VM monitoring

### Storage

- Blob Storage
- Storage security

### Security

- RBAC
- Azure Policy

### Monitoring

- Azure Monitor
- Log Analytics

## Goal

Build a strong foundation in Azure cloud architecture and infrastructure management.
